# Gold Standard Labels & Justifications

| ID | Gold Label | Justification |
|---|---|---|
| 1 | Neutral | Objective statement of fulfillment. No emotional descriptor used. |
| 2 | Negative | Explicit negative sentiment keyword ("hate"). |
| 4 | Neutral | Inquiry/question. No sentiment expressed. |
| 5 | Neutral | Observation of fact. No judgment of quality implied. |
| 6 | Neutral | User explicitly states uncertainty ("Not sure"). |
| 7 | Negative | Explicit criticism ("garbage"). |
| 8 | Negative | Outcome (unsolved issue) outweighs politeness. |
| 9 | Negative | "Not impressed" overrides "it works". |
| 10 | Neutral | Mixed sentiment (extreme positive + extreme negative) cancels to Neutral. |
| 11 | Negative | Sarcasm: "Great" and "Thanks" used ironically. |
| 12 | Positive | Product meets advertised claims. |
| 13 | Negative | Description of negative experience (billing error). |
| 14 | Neutral | Mixed sentiment ("Soft" vs "Overpowering smell"). |
| 15 | Negative | Rhetorical question implies poor design choice. |
| 16 | Positive | Explicit praise ("Incredible"). |
| 17 | Neutral | Purely informational/specs. |
| 18 | Negative | Demand for refund implies dissatisfaction/failure. |
| 19 | Negative | Comparative negative ("used to be good"). |
| 20 | Positive | Weak positive approval ("Acceptable quality"). |
| 21 | Positive | List of positive attributes. |
| 22 | Negative | Negative service experience (locked account). |
| 23 | Neutral | Functional inquiry. |
| 24 | Neutral | Treated as baseline acceptability. |
| 25 | Neutral | Balanced statement ("Not worst" vs "Not best"). |
| 26 | Neutral | Mixed sentiment ("Brilliant" vs "Terrible") cancels. |
| 27 | Negative | Fulfillment error described. |
| 28 | Neutral | Technical specification. |
| 29 | Neutral | Promotional text, no sentiment. |
| 30 | Negative | Imperative command indicating dissatisfaction. |
| 31 | Positive | Description of improvement. |
| 32 | Positive | Litotes (negating negative) implies positive. |
| 33 | Negative | Long wait times = negative service. |
| 34 | Negative | Report of a product defect/undesirable behavior (overheating). |
| 35 | Negative | Edge Case (Sarcasm): "Perfect" used to criticize the product as "broken plastic." |
| 36 | Negative | "Dealbreaker" indicates strong negative stance on a feature change. |
| 37 | Neutral | Statement of fact regarding duration. |
| 38 | Positive | Explicit recommendation. |
| 39 | Negative | "Mediocre" is a criticism of quality. |
| 40 | Neutral | Informational query. |
